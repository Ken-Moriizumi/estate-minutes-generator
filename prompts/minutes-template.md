# 議事録フォーマット

このテンプレートは、不動産賃貸業の物件検討議事録の標準フォーマットです。

## 基本構成

```
株式会社〇〇〇〇
議事録

日時：YYYY年MM月DD日 HH:MM～HH:MM
場所：[東京事務所/長野事務所/オンライン]
参加者：[役職名 氏名、役職名 氏名、...]

【議題】
1. [物件名または物件所在地]の投資検討
2. [物件名または物件所在地]の収益性分析
3. ...

【議事内容】
1. [物件名または物件所在地]の投資検討
   ＜物件情報＞
   築XX年、[住所]、購入金額X,XXX万円、表面利回りX.X%
   [その他の重要な物件情報や特記事項]

   [参加者全員の意見を完結にまとめて記載]

2. [次の議題]
   ＜物件情報＞
   ...

【結論】
[各議題の要点をまとめ、今後の方針を簡潔に記載]
- [物件A]については、[結論・次のアクション]
- [物件B]については、[結論・次のアクション]
[全体としての総括]

以上
```

## フォーマット要件

1. **日時**: 必ず「YYYY年MM月DD日 HH:MM～HH:MM」形式
2. **場所**: 「東京事務所」「長野事務所」「オンライン」のいずれか
3. **参加者**: 役職名と氏名を明記（例：「代表取締役社長 山田太郎」）
4. **議題**: 物件ごとに番号付きリストで列挙
5. **議事内容**: 各議題に対して
   - ＜物件情報＞で物件の基本情報を要約
   - 参加者ごとの発言は記載せず、全体としての結論のみ記載。
6. **結論**: 箇条書きで各物件の結論と全体総括
7. **文末**: 必ず「以上」で締める

## ボリューム

- A4用紙1〜2枚程度（800〜1,600文字）
- 物件数や議論の深さに応じて調整
